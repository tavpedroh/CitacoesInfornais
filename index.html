Sobre o Projeto

Uma Mock API é uma versão simulada de uma API real, usada principalmente para testes e desenvolvimento. Ela permite que os desenvolvedores simulem comportamentos e respostas de uma API sem precisar se conectar a um serviço de backend ativo. Isso é especialmente útil em situações como desenvolvimento paralelo, testes ou quando o backend não está disponível.
